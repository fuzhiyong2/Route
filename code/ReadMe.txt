Main.exe可直接运行，提示输入输入文件的编号。如case1，则直接输入1即可。结果会写入到output1（边格式结果）和output2（点格式结果）中
Main 编译需要c++11标准支持
struct 是所有数据结构，包含点，线段，邻接矩阵等数据结构
FileIO 是文件读取和存储操作的函数
line 是对线段和直线的各种运算关系的函数，如求线段之间的距离，点之间的距离
cor 是 判断两条直线是否相互对应的头文件函数
Scanline 是 扫描线算法
getLine 是由相互对应关系的边，得到中线线段的函数
optimize 是对getLine获得的中线的优化处理函数，包括删，合，连接等操作
tranfer 是边格式的中线转化成点格式中线使用到的函数
